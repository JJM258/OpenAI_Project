# OpenAI_Project

AI Chatbot USING LLMs (Python + Open AI API)

A conversational AI chatbot built using Python and OpenAI’s language models. This project demonstrates how to integrate Large Language Models (LLMs) into applications to generate human-like responses in real time.

Overview

This chatbot allows users to interact through a interface and receive responses. It showcases the practical use of AI in building interactive systems that can be used for customer support, virtual assistants, or recommendation engines.

## Tech
Python
OpenAI API (LLM integration)
Command Line Interface (CLI)

## Features
-  Conversational responses
-  Natural language understanding using LLMs
-  Interactive CLI-based interface
-  Extendable for different applications

## How It Works
The user enters a message via the command line
The input is sent to the OpenAI API
The language model processes the prompt
A response is generated and displayed to the user

## Installation
Clone the repository:
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
Install dependencies:
pip install openai
Set your API key:

Mac/Linux:

export OPENAI_API_KEY="your_api_key_here"

Windows (PowerShell):

setx OPENAI_API_KEY "your_api_key_here"
Run the chatbot:
python OpenAI_Chatbot.py
